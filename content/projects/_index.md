Dưới đây là các dự án tiêu biểu tôi đã thực hiện trong quá trình học tập và làm việc:

### 1. Network Programming Blog (Dự án này)
*   **Mô tả**: Blog cá nhân xây dựng bằng Hugo Static Site Generator để chia sẻ kiến thức.
*   **Công nghệ**: Hugo, PaperMod, Github Actions, Github Pages.
*   **Code**: [Link Github](https://github.com/leeT-42/LTM)


### 2. Task Management App (Android)
*   **Mô tả**: Ứng dụng quản lý công việc cá nhân trên nền tảng Android. Hỗ trợ tạo, sửa, xóa công việc, nhắc nhở và theo dõi tiến độ.
*   **Công nghệ**: Android Studio, Java/Kotlin, Firebase.

### 3 Hệ Thống Quản Lý Kho (WMS)
*   **Mô tả**: Hệ thống quản lý kho hàng chuyên nghiệp, bao gồm các chức năng cốt lõi: Nhập - Xuất - Tồn, Quản lý vị trí kho, Báo cáo thống kê và Phân quyền người dùng.
*   **Công nghệ**: ASP.NET Core (MVC/API), SQL Server, Entity Framework.
