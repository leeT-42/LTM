## 1. Let & Const
Thay thế `var` để quản lý scope tốt hơn. `const` dùng cho hằng số.

## 2. Arrow Functions
Cú pháp ngắn gọn hơn cho hàm.
```javascript
const add = (a, b) => a + b;
```

## 3. Template Literals
Chuỗi ký tự cho phép nội suy biến.
```javascript
let name = "Thanh";
console.log(`Hello, ${name}`);
```

## 4. Destructuring
Trích xuất dữ liệu từ Array/Object dễ dàng.
