Lập Trình Hướng Đối Tượng (OOP) trong Java

1. Giới thiệu Lập trình hướng đối tượng (OOP) là một mô hình lập trình dựa trên khái niệm “đối tượng”, chứa đựng dữ liệu và mã lệnh. 2. 4 Tính chất cơ bản Tính đóng gói (Encapsulation): Che giấu dữ liệu, chỉ cho phép truy cập qua method public. Tính kế thừa (Inheritance): Class con kế thừa thuộc tính/phương thức từ class cha. Tính đa hình (Polymorphism): Một hành động có thể thực hiện theo nhiều cách khác nhau. Tính trừu tượng (Abstraction): Ẩn chi tiết cài đặt, chỉ hiển thị tính năng. 3.
