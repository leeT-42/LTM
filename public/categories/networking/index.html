Status Codes 2xx (Success): 200 OK, 201 Created. 3xx (Redirection): 301 Moved Permanently. 4xx (Client Error): 400 Bad Request, 404 Not Found. 5xx (Server Error): 500 Internal Server Error.

Cơ Bản Về Mạng Máy Tính: Mô Hình OSI & TCP/IP

1. Mô hình OSI (7 tầng) Physical: Truyền dẫn vật lý (cáp, sóng). Data Link: Đóng gói dữ liệu (Frame), MAC address. Network: Định tuyến (IP address). Transport: Truyền tải tin cậy/không tin cậy (TCP/UDP). Session: Quản lý phiên. Presentation: Định dạng dữ liệu. Application: Giao tiếp với người dùng (HTTP, FTP). 2. TCP/IP (4 tầng) Thực tế sử dụng mô hình này nhiều hơn: Application Transport Internet Network Access
